# 🏨 Luxury Hotel Booking System (Full Stack)

## 📌 Project Description
The Luxury Hotel Booking System is a full-stack web application developed using:
- React JS (Frontend)
- Spring Boot (Backend)
- PostgreSQL (Database)

The application allows users to:
- Explore hotel rooms
- View amenities
- Perform bookings

It also provides backend services for:
- Managing users
- Managing rooms
- Handling reservations

---

## 🚀 Features

### 👤 User Features
- Browse hotel rooms with images and details
- View services and amenities
- Check room availability
- Book rooms online
- User authentication (Login/Register)

### ⚙️ Admin / Backend Features
- Manage rooms and hotel data
- Handle booking records
- Manage users and reservations
- REST API integration between frontend and backend

---

## 🛠️ Technologies Used

### 🎨 Frontend
- React JS
- JavaScript (ES6)
- HTML5
- CSS3 (Flexbox, Grid)

### ⚙️ Backend
- Spring Boot
- Spring Data JPA / Hibernate
- REST APIs

### 🗄️ Database
- PostgreSQL

---

## 🏗️ System Architecture
1. React frontend sends requests to backend APIs
2. Spring Boot backend processes business logic
3. PostgreSQL stores:
   - User data
   - Room details
   - Booking information

---

## 🎯 Purpose of Project
- Practice full-stack development
- Learn React component-based architecture
- Build REST APIs using Spring Boot
- Understand database design with PostgreSQL
- Learn client-server communication

---

## ⚙️ How to Run the Project

### 🔹 Backend Setup
1. Open the Spring Boot project
2. Configure PostgreSQL in `application.properties`
3. Run the Spring Boot application

### 🔹 Frontend Setup
1. Clone the repository
2. Install dependencies:
   - npm install
3. Start the application:
   - npm start
